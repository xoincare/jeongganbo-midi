# 세종실록악보 (世宗實錄樂譜) - 전폐 악곡 복원 프로젝트

## 개요

세종실록(世宗實錄) 권136~147에 수록된 악보 중, **전승이 완전히 단절된(전폐) 악곡**의 정간보 원문을 디지털화하고 MIDI로 복원하는 프로젝트입니다.

이 악곡들은 약 580년 전 세종대왕 시대에 창작되어 정간보로 기록되었으나, 임진왜란과 세월의 흐름 속에 연주 전통이 끊겨 **전 세계 어디에도 실제 소리로 복원된 음원이 없습니다.**

## 세종실록악보 구조

- **기보법**: 정간보 (井間譜) — 1행 32정간, 12대강
- **율명**: 黃(Eb4), 大(E4), 太(F4), 夾(F#4), 姑(G4), 仲(Ab4), 蕤(A4), 林(Bb4), 夷(B4), 南(C5), 無(C#5), 應(D5)
- **청성**: 潢(Eb5), 汰(F5), 㴌(G5), 㳞(Ab5) 등
- **기보 패턴** (문숙희 해독법):
  - 패턴1 (2소박): 5정간+3정간 = 같은 시간. 주로 당악 계통
  - 패턴2 (3소박): 5정간 또는 3정간 = 1박, 3소박. 주로 향악 계통

## 데이터 형식

`data/` 디렉토리에 각 악곡의 정간보를 텍스트로 디지털화:

```
# 발상지무 - 희광 (引入)
# 1행 32정간, 위→아래 순서
# 율명: 黃太仲林南無應潢汰 (빈칸 = 연장)

行1: 黃 - - - 太 - - - 仲 - - - ...
行2: ...
```

## 진행 상황

현재 원문 율명 데이터의 디지털화 작업이 필요합니다.
세종실록악보의 정간보 원문은 **원본 이미지(영인본)**로만 공개되어 있으며,
텍스트 형태의 율명 데이터는 온라인에 공개된 적이 없습니다.

### 디지털화 방법
1. 실록 영인본 원본 이미지에서 율명을 수작업 또는 OCR로 추출
2. 이혜구 역주본(1973) 참조
3. 문숙희(2025) 해독법 적용하여 리듬 해석
4. jeongganbo-midi 컨버터로 MIDI 변환
